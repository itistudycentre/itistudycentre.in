# ITI Study Centre

Official Website: https://itistudycentre.in

## 📚 About

ITI Study Centre is an educational website developed to help ITI students with free study materials, official updates, engineering notes, and result information.

## ✨ Features

- 📊 NCVT Result
- 📊 SCVT Result
- 📚 ITI Study Notes
- 🔧 Fitter Notes
- 🚰 Plumber Notes
- ⚙️ Engineering Notes
- 📰 Latest ITI News
- 🎥 YouTube Learning Support
- 📱 Mobile Friendly Design
